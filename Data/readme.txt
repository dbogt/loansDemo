Overview:
The LendingClub dataset is used for today's demos. A smaller sample has been uploaded as part of the course materials. The full data set used in the demos can be downloaded following the steps below.

Step-by-Step Guidance:
Download the LendingClub Dataset:

Visit the LendingClub Dataset on Kaggle.
Click on the "Download" button to obtain the dataset in ZIP format.
Extract the Dataset:

Locate the downloaded ZIP file (e.g., loan.zip).
Extract the contents to reveal the CSV file (accepted_2007_to_2018q4.csv).
